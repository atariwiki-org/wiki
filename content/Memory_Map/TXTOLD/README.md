||DEC||HEX||NAME||OS  
|662-667|$296-$29B|TXTOLD|both  
  
These locations contain the split-screen equivalents of [OLDROW](../OLDROW/README.md) (90; $5A), [OLDCOL](../OLDCOL/README.md) (91, 92; $5B, $5C), [OLDCHR](../OLDCHR/README.md) (93, $5D) and [OLDADR](../OLDADR/README.md) (94, 95; $5E, $5F). They hold the split-screen cursor data.
